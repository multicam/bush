# C2C: Viviana Cloud Box QuickStart Guide

> Source: https://help.frame.io/en/articles/6133587-c2c-viviana-cloud-box-quickstart-guide
> Category: c2c

---

The [Viviana Cloud Box](https://www.vivianacloud.com/) is a device you can attach to an audio recorder to upload audio files to the cloud. The Viviana Cloud Box can mount the sound recorders storage and, using the associated app, you can select files to upload. With Frame.io C2C, you can log in to Frame.io and choose a C2C Project to upload to.
​
Use this guide to quickly setup your Viviana Cloud Box for C2C with settings recommended by Frame.io. 

# What you&#x27;ll need1 Viviana Cloud Box

The latest firmware from Viviana installed on the Box

USB cable

A local network with [internet access](https://blog.frame.io/2021/03/15/connecting-your-set-to-the-internet/)The latest version of the Viviana Cloud app for iOS

A Frame.io Project with C2C Connections Integration enabled

A Frame.io account with [permissions](https://support.frame.io/en/articles/4887243-c2c-adding-and-removing-a-cloud-device) to add devices to the Project
*Specific to V3 accounts:**Ensure C2C Connections are enabled in your project settings**Ensure you have C2C Device management permissions within the account the project is hosted in*
# SetupFirst, connect your Viviana Cloud app to the Viviana Cloud Box and connect the Viviana Cloud Box to a network with internet access using [the guide](https://www.vivianacloud.com/faq/) provided by Viviana Cloud.In the Viviana Cloud app, tap the **Cloud** button on the bottom of the screen.In the Cloud menu, tap **Sign in with Frame.io**.Sign in with your Frame.io credentials and tap **Allow** to grant Viviana access to upload.Using the Project browser, select the Project you want to upload to and switch back to the Box menu by pressing the **Box** button.Finally, connect your sound recorder in file transfer mode Viviana Cloud Box via USB*.

In the Viviana Cloud app, you’ll be able to see the storage under **USB Drives**. Use this to find the files you’d like to upload and select them by tapping the checkbox next to their name.When the files are selected, tap **Upload selected files to Frame.io**. Once uploading, you will see progress at the bottom of the page.

* *You can also connect your SD, CF, or other memory card to the Viviana Cloud Box with a card reader. A USB drive can also be connected directly to the Viviana Cloud Box.* 

# Troubleshooting**How do I change the WiFi network on my Viviana Cloud Box?**Tap the power button 5 times, then use the Viviana Cloud app for iOS to connect the Viviana Cloud Box to your WiFi network.

 

**I have my recorder or storage connected to the USB-C port on the Viviana Cloud Box, but I can&#x27;t see the files. What do I do?**The USB-C port on the Viviana Cloud Box is for power only. To mount files, be sure to connect the storage to the USB-A port.

 

**How do I see my upload progress?**In the Cloud tab of the Viviana Cloud app, you will see progress for any current uploads. Completed uploads will also be marked here.
