# Inactive Projects on the iOS App

> Source: https://help.frame.io/en/articles/9958716-inactive-projects-on-the-ios-app
> Category: integrations

---

Users can access and manage inactive projects from within the iOS app. Setting a project as inactive will allow users to better organize their project visibility by filtering it out of the workspace view.
​## Setting a project to Inactive 

### From the workspaceTo set a project as inactive from the workspace, tap the **three-dot** icon on the project card to open the project settings and select “Mark Inactive”.
​### From the project pageTo set a project as inactive from the project page, tap the **three-dots** in the top right corner to access the project settings and select “Mark Inactive.
​## Accessing the Inactive ProjectsThe Workspace home page only shows active Projects.

 

 

To view all Projects in a Workspace, including any Inactive Projects, select “All Projects”. Active Projects are listed at the top and Inactive Projects are listed at the bottom.

*Note:** Enterprise users may see more than one Workspace listed on the Workspace home page.**Note:** While Inactive, projects are still fully editable and will appear still in the Project Switcher and the Move/Copy to menus. Notifications will still be sent if Inactive projects are edited.* ## Setting a project to ActiveTo change a project back to active, navigate to “All Projects” on the workspace page, tap the **three-dots** on the project and select “Mark Active”.
