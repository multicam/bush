## Overview

Frame.io is a cloud-based creative collaboration platform for video, design, and marketing teams. It provides unified workflows for file management, review & approval, sharing, and project collaboration.

**Key Metrics:**
- 2.9x faster creative workflows
- 2.7x faster review and approval (Pfeiffer Report)
- 31% reduction in review churn (IDC)

## Quick Reference

### Pricing Summary
| Plan | Price | Members | Storage |
|------|-------|---------|---------|
| Free | $0 | 2 | 2GB |
| Pro | $15/mo/member | 5 | 2TB + 2TB/member |
| Team | $25/mo/member | 15 | 3TB + 2TB/member |
| Enterprise | Custom | Custom | Custom |

### Key Features by Tier
- **Free**: C2C, Transcription, Basic collaboration
- **Pro**: Custom branding, Watermarking, Share controls
- **Team**: Internal comments, Restricted projects/folders
- **Enterprise**: SSO, DRM, Forensic watermarking, Storage Connect

### Supported Platforms
- Web
- iOS/iPadOS (native Swift)
- Apple TV (4K HDR)
- macOS (Transfer app)
- Windows (Transfer app)
- Adobe Premiere Pro
- Adobe Lightroom
- Final Cut Pro

### Security Certifications
- SOC Type 2 Compliant
- TPN+ Gold Shield Accredited
