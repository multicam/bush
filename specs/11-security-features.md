# Frame.io - Security Features

## Summary
Frame.io provides industry-leading security, scalability, and admin controls for enterprise content protection.

## Security Certifications
- **SOC Type 2 Compliant**
- **TPN+ Gold Shield Accredited**

## Content Security Features

### Watermarking

#### Static Watermarking (Pro, Team)
- One watermark template included
- Apply to shared content

#### Session-Based Watermarking (Enterprise Prime)
- Embed personal identifying information
- Viewer's email in watermark
- IP address tracking
- Geolocation data

#### Forensic Watermarking (Enterprise Prime)
- Embed invisible, unique identifiers
- Trace every view
- Trace every download
- Trace every share

### Digital Rights Management (DRM) (Enterprise Prime)
- Prevent unauthorized viewing
- Block screenshots
- Block screen recordings
- Full content protection

## Access Control

### Authentication
- Social Sign-On
- SAML2-based SSO (Enterprise)

### Share Security
- Passphrase-protected shares (Pro+)
- Share link expiration (Pro+)
- Disable asset downloads (Pro+)
- Disable comments (Pro+)
- Hide previous versions (Pro+)

### Secure Sharing (Enterprise Prime)
- Invite-only secure links
- Specific email invitations
- Manage Share access in account settings
- Secure sharing authorized domains

## Admin Controls

### Content Security Settings (Enterprise)
- Centralized security configuration
- Organization-wide policies

### Asset Lifecycle Management (Enterprise)
- Customizable retention policies
- Automatic move to trash
- Automated asset lifespan management

### Multiple Workspaces (Enterprise)
- Separate, private workspaces
- Per-client isolation
- Department separation
- Per-production workspaces
- Brand-specific workspaces

### Access Groups (Beta)
- Organize Members into groups
- Faster invites
- Advanced user management
- Permissions by group

## Storage

### Storage Connect (Enterprise Prime)
- Connect your own AWS-S3 bucket
- Direct Frame.io integration
- Control your own storage infrastructure
