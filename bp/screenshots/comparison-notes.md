# Bush vs Tailwind UI Catalyst Demo Design Comparison

Comparison of Bush UI against Tailwind UI Catalyst demo as a design quality benchmark.

## Methodology

- Bush screenshots: auto-captured via BP specs into `bush/`
- Catalyst demo screenshots: public pages auto-captured (`00-catalyst-capture.spec.ts`), authenticated views manually captured into `catalyst/`
- Each aspect rated: None (comparable), Minor (cosmetic), Major (functional impact)
