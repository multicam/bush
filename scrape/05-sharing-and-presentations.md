# Frame.io - Sharing & Presentations

## Summary
Frame.io enables users to create, customize, and control professional-looking shares with ease for beautiful and secure asset presentation.

## Unified Sharing

### Enhanced Branding
- Customize colors
- Add custom logo
- Custom titles
- Elevate the presentation experience

### WYSIWYG Editor
- Preview shares before sending
- Edit in real-time
- Send beautiful Shares to stakeholders
- All from a single view

### Viewer Options
- Curate the best player format
- Grid layout option
- Reels layout option
- Showcase work effectively

## Share Builder

### Powerfully Personalized Presentations
- Custom colors
- Custom layout
- Custom viewer configurations
- Create captivating presentations

## Secure Share Management

### Permissions and Access Controls
- View permission
- Comment permission
- Download control
- Passphrase protection
- Expiration date settings

### Enhanced Security Options
- Passphrase protection
- Expiration date control
- Advanced watermarking options

### Featured Field
- Set featured field for focused feedback
- Options: status, notes, rating, or custom
- Gather targeted responses

## Collections for Sharing

### Collaborate Better with Collections
- Tag assets with metadata
- Sort tagged assets into dynamic Collections
- Group by: status, stakeholders, due date, channel
- Build and share Collections for efficient collaboration

### Collections Drive Connections
- Assign assets to individual collaborators
- Build and share task-based Collections
- Drive clear outcomes

### Real-Time Updates
- New assets automatically update Collections
- Collections stay current automatically
- No manual refresh needed
