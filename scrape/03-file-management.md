# Frame.io - File Management

## Summary
Frame.io provides a central, secure place to upload, organize, and store all work-in-progress creative files without needing extra subscriptions.

## Upload & Transfer

### Camera to Cloud (C2C)
- Upload video, photos, audio, metadata directly from location
- No waiting for drives and cards
- Instant upload from set

### Bulk Uploads
- Upload multiple assets at once from anywhere
- Total visibility on upload progress
- Prioritize uploads

### Transfer App
- Quickly upload and download massive files
- Transfer entire projects with single click
- Folder structures remain intact
- Control transfer speeds
- Drag-and-drop to prioritize uploads
- Support for Windows and Mac
- Download options: macOS (Apple Silicon), macOS (Intel), Windows

## Intuitive Navigation

### Fast, Flexible Workspace
- Maneuver between projects and assets quickly
- Drag-and-drop simplicity
- Nested folder tree navigation

### Tailored Asset View
- Adjustable card size and aspect ratio
- Expose or hide metadata fields
- Sort by priority fields

### Multi-Panel Layout
- Expand, collapse, and navigate quickly
- View projects and assets your way
- Fluid and intuitive interface

## Metadata & Collections Integration

### Version Tracking
- Drag and drop new versions into a stack
- All asset iterations organized
- Easy reference to previous versions

### Organization
- Tag incoming assets with metadata
- Use Collections to group and sync automatically
- Stay up to date with real-time updates

## Supported File Types
- Video files
- Photos/images
- Audio files
- Documents
- Adobe suite formats
- Nearly any creative file type
